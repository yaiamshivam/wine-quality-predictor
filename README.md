# Wine Quality Prediction Website

This project is ready to run locally and deploy publicly without requiring the dataset at runtime.

## Local setup

```powershell
cd "C:\Users\Lenovo\OneDrive\Documents\2nd Semester\AI Applications"
pip install -r requirements.txt
python build_model.py
python app.py
```

Open http://127.0.0.1:5000

## Public deployment on Render

1. Upload this project to a GitHub repository.
2. Make sure `wine_quality_model.joblib` is included in the repo.
3. Create a new web service on Render and connect the GitHub repo.
4. Render will detect `render.yaml` and use:
   - Build command: `pip install -r requirements.txt`
   - Start command: `gunicorn app:app`
5. After deploy finishes, use the public `.onrender.com` URL.

## Important note

The deployed website does not need the CSV dataset for predictions. It only needs the packaged model file generated by:

```powershell
python build_model.py
```
